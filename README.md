# COVID-19 Country-Level Pro-Social Behaviour Analysis

## Overview
This repository contains the FIT3152 Assignment 1 (2024) by Fathima Saara Mohamed Ramsy.  
The project analyses country-level predictors of pro-social behaviour during the early stages of the COVID-19 pandemic, using the PsyCorona baseline dataset.  

**Achievement:** High Distinction (HD) awarded for this assignment.

The goal was to understand differences in pro-social attitudes across countries, identify important predictors, and evaluate clusters of similar countries using multivariate analysis.

---

## Dataset
- Source: PsyCorona Baseline Study (Van Lissa et al., 2022)  
- File: `PsyCoronaBaselineExtract.csv`  
- Focus country: assigned individually per student  
- Attributes include ordinal and numerical variables measuring pro-social behaviours, demographics, and lifestyle factors  

---

## Key Tasks
- Descriptive analysis and preprocessing of baseline data  
- Comparison of focus country vs all other countries  
- Clustering of similar countries using social, economic, health, and political indicators  
- Model analysis for predicting pro-social attitudes (c19ProSo01–04)  
- Creation of at least one high-quality multivariate graphic to summarize key results
